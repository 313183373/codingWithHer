# 羽毛球场管理系统

## 功能块
- 预定模块
- 取消预定模块
- 预览收入汇总模块
- 优惠（option）

### 预定模块（BadmintonManager）
- book
    - 冲突检测和合法时间段检测，如有问题，抛出异常
    - 为用户添加一条预定信息
    - 为场地schedule添加一条预定信息
    - 为场地record添加一条预定信息
- cancel
    - 用户是否存在检测和订单是否存在检测，如有问题，抛出异常
    - 删除用户的相应预定信息
    - 删除场地schedule的相应预定信息
    - 为场地record添加一条取消信息
